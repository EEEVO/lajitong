# 垃圾桶后台管理系统

## 项目简介

>此工程是基于`vue-cli3.0`为基准二次封装，适用于PC端的管理系统，初步结构与基础的工具函数已封装好

## 目录结构

```
├── README.md
├── .env环境配置文件
├── babel.config.js
├── package-lock.json
├── package.json
├── public
│   ├── favicon.ico
│   └── index.html
├── src
│   ├── App.vue     //入口文件
│   ├── assets      //全局静态资源
│   ├── components  //基础组件
│   ├── lib         //工具函数
│   ├── main.js
│   ├── plugins     //第三方依赖库
│   ├── router.js   //router相关
│   ├── store       //vuex相关
│   └── views       //视图层
└── vue.config.js   //vue-cli3.0配置文件
```
